# 迷因產生器專案規格說明
最後更新：2025年5月26日

## 專案概述
這是一個純前端的迷因（Meme）產生器網站，設計用於在瀏覽器中運行，並且可以直接部署到 GitHub Pages 上。專案使用純 HTML、CSS 和 JavaScript 開發，外部依賴採用 CDN 方式引入。

## 專案目標
1. 建立一個簡單易用的迷因產生器
2. 支援各種圖片編輯和文字處理功能
3. 提供良好的使用者體驗
4. 確保跨裝置相容性

## 技術堆疊
- HTML5 (包含 Canvas API 用於圖片處理)
- CSS3 (用於頁面樣式和響應式設計)
- JavaScript (純原生，無框架，用於核心功能實現)
- Bootstrap 5.3.0 (通過 CDN 引入，用於 UI 元件和響應式布局)

## 核心功能
1. 圖片上傳與處理
   - [x] 支援上傳本地圖片
   - [x] 自動調整圖片大小以適應畫布
   - [ ] 支援圖片裁剪功能
   - [ ] 支援拖拽上傳

2. 文字編輯
   - [x] 支援上方文字輸入
   - [x] 支援下方文字輸入
   - [x] 文字自動轉換為大寫
   - [x] 可自訂文字顏色
   - [ ] 可調整文字大小
   - [ ] 可選擇文字字體
   - [ ] 支援文字拖拽定位

3. 迷因生成
   - [x] 即時預覽
   - [x] 支援下載為 PNG 格式
   - [ ] 支援其他圖片格式（如 JPG）
   - [ ] 支援調整輸出圖片品質

4. 使用者介面
   - [x] 響應式設計（使用 Bootstrap）
   - [x] 簡潔的卡片式布局
   - [ ] 深色模式支援
   - [ ] 多語言支援

## 程式碼結構
### 主要文件
- `index.html`: 主要的 HTML 文件
  - 包含頁面結構和 UI 元素
  - 使用 Bootstrap 5.3.0 進行布局
  - 包含所有必要的 UI 控制元件

- `style.css`: 樣式表文件
  - 定義自訂樣式
  - 處理 Canvas 和卡片的視覺效果
  - 實現響應式設計的補充樣式

- `script.js`: JavaScript 核心邏輯
  - 圖片上傳和處理邏輯
  - Canvas 繪圖實現
  - 文字處理和定位
  - 下載功能實現

### 代碼風格規範
1. JavaScript
   - 使用最新的 ES6+ 語法
   - 採用事件委派模式處理事件
   - 使用 async/await 處理異步操作

2. CSS
   - 優先使用 Bootstrap 類別
   - 自訂樣式採用 BEM 命名約定
   - 使用 CSS 變數管理主題顏色

3. HTML
   - 保持語意化標籤使用
   - 確保無障礙性（ARIA 屬性）
   - 保持結構清晰的縮排
